Robot Dimensions
!
If there is only one input, then the value of the other input is automatically the value of another input. This is to avoid Errors or to easily create a square dimension quick since square robots are quite common.
Max Speed
!
To get the Max Speed, You need to program the robot to move forward in a straight line for a second and measure the distance it travelled. This part is crucial, It is one of the base values of almost all of the formulas I used, If you got this wrong then all generated data are wrong.
Wheel Distance
!
The Wheel Distance is the distance between the middle of the left wheel to the middle of the right wheel. Meausuring this will create the anchor point in between that is used for the rotation of the robot
!
Unit used are not necessary to be in centimeters or meters, As long as you use only one type of unit then you don't have to worry about mis-calculations.

Centimeters is a common unit that is usually used for small robots, which this project was supposed for.
